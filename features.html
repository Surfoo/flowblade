Features

Workflow

  • Toolset with 6 edit tools
  • Compositing Mode can be selected per sequence to be either Standard Full Track or Top Down Free Move.
  • Many timeline behaviours are configurable

Edit tools

  • Move
  • Insert
  • Spacer
  • Multitrim
  • Cut
  • Keyframe

Timeline features

  • Insert, Append, Overwrite Range and Overwrite Clip actions for adding clips on the Timeline
  • All compositors keyframeble
  • Most filters keyframeble
  • Drag and drop clips on the Timeline
  • Clip and Compositor parenting and resync with other clips
  • Split audio from video
  • Audio levels display
  • Clone and copy paste clips and filters
  • Mute track video or audio
  • Named markers
  • Max 9 combined video and audio tracks available
  • Change tracks count
  • JKL playback and advanced arrow key seek
  • Slow motion and fast speed clips
  • Export: MLT XML, Adrour session, Frame image

Compositors

  • Affine Blend
  • Blend
  • Transform
  • Alpha: LumaToAlpha, AlphaXOR, AlphaOut, AlphaIn
  • Wipe: Wipe/Translate, Wipe Clip Length
  • Blend: 18 different blend mode Compositors
  • Fades: Fade In, Fade Out

Filters

  • Alpha: Alpha Gradient, Alpha Modify, Alpha Shape, Chroma Key, Color Select, Crop, Luma Key, Spill Supress
  • Artistic: Cartoon, Charcoal, Color Halftone, Dither, Emboss, Glow, Old Film, Scanlines, Soft Glow, Vignette
  • Audio: Mono to Stereo, Pan, Pan Keyframed, Swap Channels, Volume
  • Audio Filter: Bandpass, Chorus - Multivoice, Declipper, Delayorama, Distort - Barry's Satan, Distort - Diode Processor, Distort - Foldover, Distort - Pointer cast, Distort - Sinus Wavewrap, Equalize - DJ 3-band, Equalize - Multiband, Flanger - DJ, Frequency Shift - Bode/Moog, GSM Simulator, Highpass - Butterworth, Lowpass - Butterworth, Noise Gate, Pitchscaler - High Quality, Pitchshifter - AM, Rate Shifter, Reverb - GVerb, Reverb - Plate, Signal Shifter, Vinyl Effect
  • Blur: Blur, Box Blur, Grain, IRR Blur, Pixelize, RGB Noise
  • Color: Brightness, Color Adjustment, Color Grading, Color Lift Gain Gamma, Contrast, Curves, Gamma, Grayscale, Hue, Invert, Levels, Lift Gain Gamma, RGB Adjustment, Saturation, White Balance
  • Color Effect: Chroma Hold, Color Clustering, Color Distance, Color Tap, Colorize, Newspaper, Posterize, Primaries, Sepia, Technicolor, Three Layer, Threshold, Threshold0r, Tint
  • Distort: Flip, Image Grid, Lens Correction, Mirror, V Sync, Waves
  • Edge: Edge Glow, Sobel
  • Fix: Denoise, Letterbox, Sharpness
  • Movement: Baltan, Freeze, Nervous, Vertigo
  • Transform: Affine, Rotate, Shear, Translate

Range Log

  • Flowblade provides functionality to save, name and manage in point to out point ranges on media items
  • This is useful e.g. for media items that contain multiple areas of interest
  • Drag and drop items/clips from/to Range Log panel
  • Arrange Range Log items in groups and sort by file name, time or comment
  • Add selected items to Timeline in a single action

Proxy Editing

  • Proxy editing is a method of editing in which original media clips are presented on timeline by proxy clips
  • This is needed when original media makes too high demands for either disk bandwidth or CPU processing power to enable responsive editing
  • User can freely change between original media and proxy media

Batch Render Queue

  • Flowblade offers a dedicated Batch Render Queue application. Batch Render Queue is a separate application to Flowblade and runs on different process, so it is possible to close Flowblade without affecting ongoing renders
  • Render queue is a persistent data structure of render items on disk. Each item consists of a Project file and saved render parameters. Users can add render items to render queue and then render the whole queue without any further user input

G'Mic Effects tool

  • G'MIC is a full-featured open-source framework for image processing
  • Flowblade G'MIC tool presents user with a selection of commads that can be edited and combined to achieve complex filtering of video clips

Audio Mixer

  • Monitor audio levels on VU meters during playback
  • Set volume for tracks or Master Out
  • Set pannig for tracks

Media Relinker

  • Media Relinker is a standalone application running in its own process and does not affect the project that is open in Flowblade
  • Replace missing media with other media of your the choice
  • Save fixed projects for further editing

Titler

  • Multiple text layers
  • Position text using image of background video
  • Set font, color size and style separately for all text layers

Misc. Features

  • USB JOb/Shuttle support
  • Slow motion and fast speed clips
  • Watermarks

Rendering
